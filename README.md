#  Flutter Gelişim Günlüğüm: İlk Uygulamam

Flutter öğrenme yolculuğumun ilk meyvesi olan Sevgililer Günü Kartpostalı projesi! Bu proje ile mobil geliştirme dünyasına adım attım.

##  Neler Öğrendim?
* `void main()` ve `runApp()` ile uygulama başlatma sürecini.
* `MaterialApp` ve `Scaffold` temel widget yapılarını kullanmayı.
* `AppBar` tasarımı yapmayı ve başlığı ortalamayı (`centerTitle: true`).
* `TextStyle` ile font ailesini (`güzelYazi`), boyutunu ve rengini özelleştirmeyi.
* `Center` ve `Image.asset` kullanarak ekrana yerel görseller eklemeyi.
